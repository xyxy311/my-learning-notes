# 术语
数据：由数据元素组成
数据元素：数据的基本单位，数据的个体
数据项：构成数据元素的最小单位
数据对象：性质相同的数据元素的集合，数据的子集

**数据结构**：数据元素之间的关系
- 逻辑结构
	- 线性结构
	- 非线性结
	另一种分类：
	- 图
	- 树
	- 链
	- 集合
- 存储结构
	- 顺序
	- 链接存储结构：存储元素的同时存储下一个元素的地址
	- 索引：存储数据元素时建立索引表
	- 散列
- 数据的运算和实现
