# 数组

一维数组可以看作线性表，二维数组可以看作以线性表为数据元素的线性表
```cpp
typedef Elemtype array[m][n];
"等价于："
typedef Elemtype array_e[n];
typedef array_e array[m];
```

## 存储结构
### 头尾链表

需要两种节点，tag表示是否为表
![[Pasted image 20251226082047.png]]

```cpp
typedef enum{ATOM, LIST} ElemTag;  // 枚举

typedef struct GLNode {
	ElemTag tag;
	union {
		AtomType atom;
		struct {struct GLNode *hp, *tp;} ptr;
		// ptr 为指针域, ptr.hp：头指针
	}
} *GList;
```
