要包含头文件：
	`<fstream>`读写
	`<ifstream>`读
	`<ofstream>`写
文件分类：
- 文本文件
- 二进制文件

# 1 文本文件

##  读写
创建流对象 --> 打开文件 --> 写入 --> 关闭文件
![[Pasted image 20251127142154.png]]
```cpp
#include<iostream>
#include<fstream>
using namespace std;

int main(){
    ofstream ofs;
    ofs.open("text1.txt",ios::out);
    ofs<<"XiaoMing"<<endl;
    ofs<<"XiaoShuai"<<endl;
    ofs.close();
    cout<<"OK";
    
    return 0;
}
```

```cpp
#include<iostream>
#include<fstream>
#include<string>
using namespace std;

int main(){
    ifstream ifs;
    ifs.open("text1.txt",ios::in);
    if(ifs.is_open())  //确定文件是否打开成功
        cout << "OK" << endl;
    
    char buf[1024] = {0};
    while(ifs>>buf)  //不读取空白符，类似scanf
        cout << buf << endl;
    while(ifs.getline(buf,1024))  //读取一行，包括空白
        cout << buf << endl;
    string s;
    while(getline(ifs,s))   //读取一行，包括空白
        cout << s << endl;
    char c;
    while((c = ifs.get()) != EOF) //逐个字符读取，包括换行符
        cout << c;
    ifs.close();

}
```

# 2 二进制文件
使用`ofs.write(const char* buffer , len)`写文件
`buffer`:写入数据的地址
`len`:写入长度

使用`ofs.read(char* buffer , len)`写文件
`buffer`：写入的位置指针
