# Lambda表达式

```cpp
[捕获变量](参数列表) 可选限定符-> 返回类型 {
	函数体
};

// 例子一
int x = 10;
int y = 3;

auto p = [x, y](int a, int b) {
	return x * a + y * b;
};

cout << p(3, 4);

// 例子二
sort(v.begin(), v.end(), [](int x, int y){return x > y;});
```

`[x, y]`捕获变量时是值拷贝，创建函数对象后，修改被捕获的变量，不影响捕获变量的值
`[&x, &y]`表示按引用捕获，修改被捕获的变量，影响捕获变量的值

值捕获时，lambda对应的调用函数默认是const函数
引用捕获时，不是const函数

可以指定默认捕获方式和特例
```cpp
auto p = [=, &y](int a, int b) {
	// x++;  不允许
	y++;  // 可行
	return x * a + y * b;    // 默认是值捕获，对y进行引用捕获
};
```

可以捕获`this`，`[this]`：引用捕获，`[*this]`：值捕获
