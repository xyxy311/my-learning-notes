# 1.1 程序运行之前：
编译之后，执行之前，分为两个区域
**代码区**：
	存放二进制机器指令
* 共享
* 只读
**全局区**：
	存放全局变量 静态变量 常量
	在程序结束后，由操作系统释放
```cpp
#include "iostream"
using namespace std; 

int a = 10;
int b = 23;
const int c =23;
int main()
{
	int x=3;
	int y=4;
	const int z =5; 
	static int p = 10;
	static int q = 10;

	cout<<&a<<endl; 
	cout<<&b<<endl;
	cout<<&x<<endl;
	cout<<&y<<endl;
	cout<<&p<<endl;
	cout<<&q<<endl;
	cout<<&"Hello"<<endl;
	cout<<&c<<endl;
	cout<<&z<<endl;
	/*
0x472010  a   全局区
0x472014  b   全局区
0x70fe4c  x
0x70fe48  y
0x472018  p   全局区
0x47201c  q   全局区
0x488001  "Hello"   全局区
0x488008  c   全局区
0x70fe44  z
	*/
} 
```
# 1.2 程序运行之后
**栈区**：
	由编译器自动分配释放，存放形参 局部变量
**堆区**：
	由程序员自行管理，程序结束后系统强制回收
	使用`new`和`delete`运算符开辟堆区内存
```cpp
#include "iostream"
using namespace std; 

int* a()
{
	//new在堆区上开辟内存存放数据，返回指针（该内存的地址） 
	//分配失败则返回 NULL 指针
	int* p = NULL;
	p = new int(10);
	return p; 
}
int main()
{
	int* p = a();
	cout<<*p;  //输出10
	delete p;  //释放p指向的内存
} 
```
在数组中使用：
* 一维数组：
```cpp
// 动态分配,数组长度为 m
int *array=new int [m];
//释放内存
delete [] array;
```
* 二维数组：
```cpp
int **array;
// 假定数组第一维长度为 m， 第二维长度为 n
// 动态分配空间
array = new int *[m];
for( int i=0; i<m; i++ )
{
    array[i] = new int [n];
}
//释放
for( int i=0; i<m; i++ )
{
    delete [] array[i];
}
delete [] array;
```
