# const的使用

“**能加 const 就加 const**” —— 这是 C++ 编程的重要原则，有助于编写更安全、更清晰、更高效的代码。

| 场景       | 推荐写法                   | 目的                                    |
| -------- | ---------------------- | ------------------------------------- |
| **传入大对象**    | `void f(const T&)`     | 避免拷贝 + 防修改                            |
| **成员函数不改状态** | `T get() const`        | 允许 const 对象调用，语义清晰                    |
| **返回内部数据**   | `const T& get() const` | 防止外部篡改内部状态                            |
| **操作符重载**    | 提供 const 和非 const 版本   | 支持灵活使用                                |
| **字符串/数组参数** | `void f(const char*)`  | 兼容字符串字面量（如 `"hello"` 是 `const char*`） |
